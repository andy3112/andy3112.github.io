Lectura remota de medidores de energía eléctrica

Diseño y desarrollo Hardware y Software de dispositivo electrónico capaz de obtener datos eléctricos de medidores de energía eléctrica. Este dispositivo recolecta la información, y luego lo envía a un concentrador utilizando tecnología Lora. Esta tecnología permite cubrir grandes distancias con un consumo mínimo. Al final los datos se almacenan en un servidor y se pueden acceder y visualizar en tiempo real desde cualquier dispositivo conectado a internet. Con esta información se puede tener con mucha precisión el consumo eléctrico, graficas de carga eléctrica, y detectar problemas como fugas, robos ... etc.

Sensores, Actuadores y controlador Lora

Diseño y desarrollo de dispositivos IOT: sensores, controlador y actuadores, en un sistema IOT con tecnología Lora para controlar la cantidad de riego necesario para cultivos, los sensores Lora permiten obtener datos en tiempo real de humedad y conductividad del suelo, lo que permite controlar adecuadamente la cantidad de agua que los sistemas de riego ya sea ppr aspersión o goteo necesitan para optimar recursos y mejorar la calidad de los productos en diferentes tipos de cultivos.

Esta herramienta tecnológica nos permiten experimentar con nuevos diseños y formas, que incluso mediante técnicas tradicionales serian imposibles de fabricar.
